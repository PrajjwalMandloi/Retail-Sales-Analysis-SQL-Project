

# Retail Sales Analysis SQL Project

## About The Project

I built this project to practice SQL concepts and understand how sales data can be analyzed using a relational database.

The project contains customer, product, order, and sales information. 
Using SQL queries, I analyzed customer spending patterns, product performance, category-wise revenue, and city-wise sales trends.


## Database Tables

- Customers
- Products
- Orders
- Order_Items

These tables are connected using primary and foreign keys.


## Tools Used

- MySQL Workbench
- SQL
- Python (for generating sample data)


## What I Worked On

- Database creation
- Table relationships
- Data generation using Python
- Data analysis using SQL queries
- Revenue analysis
- Customer analysis
- Product analysis
- ER Diagram creation


## SQL Concepts Used

- Joins
- Aggregate Functions
- Group By
- Order By
- Views
- Stored Procedures
- Indexing
- Window Functions


## Business Questions Solved

- Who are the top customers by spending?
- Which products are sold the most?
- Which category generates the highest revenue?
- Which cities contribute the most sales?
- What are the overall sales trends?


## Key Findings

- Accessories generated the highest revenue.
- Delhi contributed the highest sales revenue.
- Keyboard was one of the most sold products.
- A small group of customers contributed a significant portion of total revenue.
